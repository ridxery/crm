### Installation

 Clone the repo
   ```sh
   git clone https://github.com/ridxery/crm
   ```
 Install Composer/PHP packages
   ```sh
   composer install
   ```
 Copy example .env file
   ```sh
   cp .env.example .env
   ```
 Generate Key
   ```sh
   php artisan key:generate
   ```
 Migrate Database with seeders
   ```sh
   php artisan migrate:refresh --seed
   ```
